# atom-config
This repo is intended to document my atom preferences/config so I can easily pull them down on a new machine when needed.

## To Use
1. Install seti-ui theme and select it
2. Select One Dark syntax
3. Copy styles.less
4. Install packages
  1. atom-beautify
  2. atom-typescript
  3. git-plus
  4. jshint
  5. language-puppet
  6. linter
